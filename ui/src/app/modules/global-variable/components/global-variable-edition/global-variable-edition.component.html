Global variable can be user in your scenario or components with **global_variable_name**.
You can defined variable in json in this file.
For example :
{{"{
        key1: value1,
        key2: {
                subKey1: subValue1,
                subKey2: subValue2
        },
        key3: [
                {
                alias: nico,
                test:value
                },
                {
                test:value
                }
        ]
}"}}
will generate keys :
        - subKey1
        - key2.subKey1
        - key2.subKey2
        - nico.test
        - key3[1].test
