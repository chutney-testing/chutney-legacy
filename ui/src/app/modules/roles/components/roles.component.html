Declare Roles and Map users to them.
            A role DEFAULT has to be declared.

            Roles has to be declared in a "roles" objects' list.
            Role template :
            {{'{
                "name": "ROLE_NAME",
                "rights": [
                    "SCENARIO_READ",
                    "SCENARIO_WRITE",
                    "SCENARIO_EXECUTE",
                    "CAMPAIGN_READ",
                    "CAMPAIGN_WRITE",
                    "CAMPAIGN_EXECUTE",
                    "ENVIRONMENT_ACCESS",
                    "GLOBAL_VAR_READ",
                    "GLOBAL_VAR_WRITE",
                    "DATASET_READ",
                    "DATASET_WRITE",
                    "COMPONENT_READ",
                    "COMPONENT_WRITE",
                    "ADMIN_ACCESS"
                ]
            }'}}

            Users-roles mapping has to be declared in a "authorizations" objects' list.
            User-Role mapping template :
            {{'{
                "name": "ROLE_NAME",
                "users": [
                    "user-id-1",
                    "user-id-2"
                ]
            }'}}
